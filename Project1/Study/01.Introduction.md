## 자바 기초 문법의 이해
## IDE : eclipse, STS(Spring Tool Suite), IntelliJ 
## 수업내용
> 객체지향 프로그래밍 <br>
> Maven - 프로젝트 관리 도구 (Build tool) <br>
> Java를 이용한 CRUD <br>
> 파일 입출력 <br>
> 데이터 베이스 : SQLite <br>
> 기본 SQl (insert, update, delete, select 문) <br>
> JDBC로 데이터베이스를 이용한 데이터 관리 <br>
> MyBatis framework 사용한 데이터 관리 <br>
